# Nanopilot Privacy Policy

Nanopilot does not collect, store, transmit, or share any personal user data.

All user settings (server host, port, token secret) and chat history are stored exclusively on the user's local device via `chrome.storage.local`. No data is sent to any third-party server other than the user-configured Nanobot backend for the purpose of chat communication.

The extension does not use analytics, tracking, cookies, or any form of third-party data collection.

For questions, please open an issue on [GitHub](https://github.com/chengyongru/Nanopilot/issues).
